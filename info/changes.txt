فتح بالمتصفح: لِفتح الإضافة بمتصفح الويب, أَي يذهب إلى صفحى الإضافة مِن موقع إضافات nvda.
نسخ رابط التنزيل: لِنسخ رابط مباشر لِتنزيل الإضافة للحافِظة.
نسخ رابط صفحة الإضافة: لِنسخ رابط صفحى الإضافة على موقع إضافات nvda.

As for the new modifications / additions.
* Solve the problem of installing the program on disk C:, now you can install it without having any problems.
* A new tool has been added, through which you can download nvda add-ons, you can access it by pressing nvda+shift+n, or from the Tools menu, choose the download nvda addons tools ctrl+shift+n
After entering the tool, a list of addons from the nvda site will appear
https://addons.nvda-project.org
Note, only addons that support direct download links are available in the program.
As for the details of using this tool.
To download an addon, just press enter, and it will start the download, and tell you the download percentage via the progress bar in the window.
As for the download path, the default is in Documents, and you can change the path in settings/download nvda addons tool settings
After identifying the addon, you can press F1, to find out information about the addon, the required version of NVDA to run the add-on, as well as the description of the addon.
Note, the required NVDA version number may be inaccurate in the future, because the description and version are stored data, i.e. manually changed, and the addons version is always up-to-date, i.e. always download the latest available version.
There are some options in the context menu after stopping the addition, which are:
Open: To open the addon, this option appears if the addon has already been downloaded.
Show in folder: To open the folder that contains the addon and move the cursor to the addon directly, this option appears if the addon has already been downloaded.
Open with browser: To open the addon in a web browser, that is, go to the addon page of the nvda addons website.
Copy Download Link: To copy a direct link to download the addon to clipboard.
Copy addon Page Link: To copy the addon page url on the nvda addons website.
That's all about this tool.
*Added a new tool, random password generator.
You can access it by pressing control+shift+p, or from the Tools menu select ctrl+shift+p random password generator.
As for how to use it, it is very easy, after entering it, we specify the length of the password to be generated, 4/170
We specify the desired options, such as including capital letters, symbols, and numbers, then press generate, it will take us to the edit field that contains the password, we can copy it by selecting control+a and then copying control+c.
